# 🤖 AI Chatbot using Python & Groq API

A smart AI-powered chatbot built using Python and the Groq API.
This chatbot can understand user messages and generate intelligent real-time responses using LLM models.

---

# 🚀 Features

* Real AI conversation
* Fast responses using Groq API
* Continuous chat support
* Easy to use
* Beginner friendly
* Python based

---

# 🛠 Technologies Used

* Python
* Groq API
* VS Code

---

# 📦 Installation

## 1. Clone the repository

```bash
git clone <your-repository-link>
```
---

## 2. Install dependencies

```bash
pip install groq
```

---

# 🔑 Get Groq API Key

1. Visit:

https://console.groq.com/keys

2. Create a new API key

3. Copy your API key

---

# ▶️ Run the Project

```bash
python main.py
```
